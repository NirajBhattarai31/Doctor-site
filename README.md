# 🏥 Doctor Site (Dockerized React + Vite App)

A modern healthcare website built using React + Vite and deployed using Docker with NGINX and Docker Compose.

---

## 🚀 Tech Stack

- React (Vite)
- Tailwind CSS
- Docker
- NGINX
- Docker Compose
- Git & GitHub

---

## 🐳 DevOps Features

- Multi-stage Docker build (Node → NGINX)
- Production-ready static hosting
- Containerized deployment
- Docker Compose orchestration

---

## 📦 How to Run Locally

```bash
docker-compose up --build
```
